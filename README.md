# 知识库

将原始 AI 研究材料转化为结构化、持续演进的 Markdown 知识库。

领域：**高性能计算（HPC）**、**AI 基础设施**、**AI Agent**。

## 目录结构

```text
raw/          源材料。仅追加，永不编辑现有文件。
  assets/     图片资源（Git LFS 存储）
wiki/         编译后的知识页面
  concepts/   原子知识单元
  topics/     更广泛的主题概述
  summaries/  每源摘要 — 摄入状态存于此
output/       临时暂存，永不提交
schemas/      Markdown 模板
```

## 使用方法

### 在 Obsidian 中浏览

1. 打开 Obsidian
2. 将此目录作为仓库打开
3. 从 `wiki/index.md` 开始导航

### 搜索知识

使用 Obsidian 的搜索功能（Ctrl/Cmd + Shift + F）搜索关键词。

### 添加新知识

1. 将源材料放入 `raw/` 目录
2. AI 助手会自动编译到 `wiki/`
